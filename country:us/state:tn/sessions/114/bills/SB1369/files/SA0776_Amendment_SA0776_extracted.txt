Section 2:
SECTION __. Tennessee Code Annotated, Section 68-11-1602(10)(B), is amended by
adding the following as a new subdivision:
( ) An agency, institution, facility, or place, whether publicly or privately owned
or operated, that provides cardiac catheterization;

Section 3:
SECTION __. Tennessee Code Annotated, Section 68-11-1607(a)(3)(E), is amended by
deleting the subdivision.

Section 4:
SECTION __. Tennessee Code Annotated, Section 68-11-204(a)(1), is amended by
adding the following as a new subdivision:
( ) Cardiac catheterization;

Section 5:
SECTION __. Tennessee Code Annotated, Section 68-11-204, is amended by adding
the following as a new subsection:
( )
(1) A person or entity shall not initiate or provide cardiac catheterization
services unless the facility providing such services is licensed as an outpatient
department of a hospital licensed under this title.
(2) Subdivision ( )(1) does not apply to a facility providing cardiac
catheterization services that was in operation as of the effective date of this act.

Section 6:
SECTION __. Tennessee Code Annotated, Section 68-11-202(a)(1), is amended by
adding "cardiac catheterizations," immediately preceding "recuperation centers".
