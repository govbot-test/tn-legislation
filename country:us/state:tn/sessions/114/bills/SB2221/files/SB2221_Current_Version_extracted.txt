Section 1:
By Reedy
SENATE BILL 2221
By Yager
AN ACT to amend Tennessee Code Annotated, Title 71,
Chapter 5, Part 15, relative to ambulances.
BE IT ENACTED BY THE GENERAL ASSEMBLY OF THE STATE OF TENNESSEE:

Section 2:
SECTION 1. Tennessee Code Annotated, Section 71-5-1506(a), is amended by
deleting the language "fifty dollars ($50.00) per calendar day for each day" and substituting
instead the language "five percent (5%) of the principal amount owed per month for each
month".

Section 3:
SECTION 2. Tennessee Code Annotated, Section 71-5-1509, is amended by deleting
the language "June 30, 2026" and substituting instead the language "June 30, 2027".

Section 4:
SECTION 3. This act takes effect upon becoming a law, the public welfare requiring it.

Raw Text:
HOUSE BILL 2097
By Reedy
SENATE BILL 2221
By Yager
AN ACT to amend Tennessee Code Annotated, Title 71,
Chapter 5, Part 15, relative to ambulances.
BE IT ENACTED BY THE GENERAL ASSEMBLY OF THE STATE OF TENNESSEE:
SECTION 1. Tennessee Code Annotated, Section 71-5-1506(a), is amended by
deleting the language "fifty dollars ($50.00) per calendar day for each day" and substituting
instead the language "five percent (5%) of the principal amount owed per month for each
month".
SECTION 2. Tennessee Code Annotated, Section 71-5-1509, is amended by deleting
the language "June 30, 2026" and substituting instead the language "June 30, 2027".
SECTION 3. This act takes effect upon becoming a law, the public welfare requiring it.
