AMEND S e n ate Bill No. 2537 House Bill No. 2207*
by deleting all language after the enacting clause and substituting:
SECTION 1. Tennessee Code Annotated, Section 67-5-212(b)(3)(A)(i), is amended by
deleting the language "thirty (30) days" and substituting "one hundred eighty (180) days".
SECTION 2. Tennessee Code Annotated, Section 67-5-212(b)(3)(A)(ii), is amended by
deleting the language "thirty (30) days" and substituting "one hundred eighty (180) days".
SECTION 3. Tennessee Code Annotated, Section 67-5-603(d), is amended by deleting
the language "governing body" and substituting "governing body by resolution".
SECTION 4. Tennessee Code Annotated, Section 67-5-606(c), is amended by deleting
the language "governing body" and substituting "governing body by resolution".
SECTION 5. Tennessee Code Annotated, Section 67-5-1512(c), is amended by
deleting the period at the end of the sentence and adding the words ", updated annually each
subsequent year.".
SECTION 6. Tennessee Code Annotated, Section 67-5-1512(d)(1), is amended by
adding the language ", updated annually each subsequent year" immediately after the language
"points".
SECTION 7. Tennessee Code Annotated, Section 67-1-401(a)(6)(A), is amended by
deleting the language "seven (7)" and substituting "twelve (12)".
SECTION 8. Tennessee Code Annotated, Section 67-1-401(a)(6)(B), is amended by
deleting the language "seven (7)" and substituting "twelve (12)".
SECTION 9. This act takes effect upon becoming law, the public welfare requiring it.
