AMEND S e n ate Bill No. 181* House Bill No. 288
by deleting all language after the enacting clause and substituting:
SECTION 1. Tennessee Code Annotated, Section 4-29-245(a), is amended by deleting
subdivision (3).

the following new subdivision:
( ) Interstate Medical Licensure Compact, created by § 63-6-401;
SECTION 3. Tennessee Code Annotated, Section 4-29-247(a), is amended by inserting
the following new subdivision:
( ) Tennessee child fatality prevention team, created by § 68-142-102;
SECTION 4. Tennessee Code Annotated, Section 4-29-248(a), is amended by deleting
subdivision (19).
SECTION 5. Tennessee Code Annotated, Section 8-34-209, is amended by deleting the
section.
SECTION 6. Tennessee Code Annotated, Section 58-7-102(b), is amended by deleting
the language "All appointments shall be subject to review by the joint select committee on
veterans services of the general assembly.".
SECTION 7. Tennessee Code Annotated, Section 58-7-109, is amended by deleting the
language "and the joint select committee on veterans services".
SECTION 8. Tennessee Code Annotated, Section 68-11-267, is amended by deleting
the section.
SECTION 9. This act takes effect upon becoming a law, the public welfare requiring it.
